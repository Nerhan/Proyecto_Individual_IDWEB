# SubSónica 

**URL del proyecto:** [https://subsonico.onrender.com/](https://subsonico.onrender.com/)

## Descripción del Proyecto
SubSónica es una plataforma digital diseñada para centralizar y difundir la escena musical independiente, permitiendo a los usuarios descubrir y registrar información detallada sobre bandas y canciones. A través de una interfaz dinámica y perfiles personalizados, el proyecto actúa como un motor de visibilidad cultural que preserva la memoria artística local y facilita la conexión entre los músicos y su audiencia en un catálogo organizado y accesible.

## Lenguajes y Tecnologías Utilizados
Para el desarrollo integral de esta aplicación web, se emplearon los siguientes lenguajes y herramientas:

* **HTML5**: Estructuración de las páginas y formularios de registro.
* **CSS3**: Diseño visual, estilos de los perfiles y adaptabilidad (Responsive Design).
* **JavaScript (JS)**: Lógica del lado del cliente y consumo de la API interna para la carga dinámica de datos.
* **Python**: Lenguaje principal del backend, utilizando el framework **Flask** para la gestión de rutas y lógica del servidor.
* **SQL**: Lenguaje de consulta para la gestión y persistencia de datos dentro de la base de datos relacional (**SQLite**).
